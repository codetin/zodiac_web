How it works

The Apollo Project ("TAP") operates a Blockchain based Decentralized Online Entertainment (DOE) Platform we call the dVerse.

dVerse Design

Manages Security and Global Login. Implements the basic functionality needed for The dVerse to interface with underlying Technologies such as Ethereum, On-Chain Storage and NoSQL (Off-Chain Storage). Provides MUSE functionality such as Trading and Wallet (through partner relationships). Provides higher level functions such as Avatars, Breeding, Crafting, Chat and many others.

dVerse Avatars

The dVerse provides a fully functioning and high-quality artistic Avatar system. This is the driving force behind blockchain gaming. Avatars are in the dVerse. All Games in the dVerse can use and add to the dVerse Avatar system. The Generation Zero (Gen 0) Art for this system is based on the Zodiac from both Asian and Western Cultures.

dVerse Games

All TAP Games will add content, functions and modes after their initial release. Nothing stays static. Players will be able to take their dVerse Avatar into each game. Gaining Experience Points and Leveling Up an Avatar in a game carries over into the Platform and other games. Game Objects can often be re-used from one game to the next.
